  **Предусловия:**
  - Пользователь авторизован в системе
  - У пользователя есть необходимые права доступа

  **Основной сценарий:**
  1. Пользователь открывает форму/страницу
  2. Система отображает интерфейс с данными
  3. Пользователь выполняет действие (нажимает кнопку, вводит данные)
  4. Система обрабатывает запрос
  5. Система отображает результат пользователю

  **Альтернативный сценарий:**
  - Если данные некорректны — система показывает сообщение об ошибке
  - Если нет прав доступа — система запрещает действие

  **Ограничения:**
  - Максимальное количество записей за операцию: N
  - Доступно только для роли X
  - Время выполнения не более Y секунд
