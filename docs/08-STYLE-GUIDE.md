# Руководство по стилям (Style Guide)

## Обзор

Генератор ЧТЗ использует стили из файла шаблона `templates/gpn-template.docx`. Это позволяет изменять оформление документов без редактирования кода.

## Файлы шаблонов

| Файл | Назначение |
|------|------------|
| `gpn-template.docx` | Основной шаблон (колонтитулы + стили) |
| `gpn-template-styleguide.docx` | Style guide с примерами всех элементов |

## Что определяется в шаблоне

### Стили параграфов (styles.xml)

| Стиль | styleId | Используется для |
|-------|---------|------------------|
| Heading 1 | `1` | Заголовки разделов (# 1. Термины...) |
| Heading 2 | `2` | Подзаголовки (## 1.1 Подраздел) |
| Heading 3 | `3` | Под-подзаголовки (### 1.1.1) |
| Normal | `a` | Обычный текст |
| List Paragraph | `aff2` | Элементы списков |
| Hyperlink | `aff5` | Гиперссылки |

### Форматы нумерации (numbering.xml)

- Маркированные списки (•)
- Нумерованные списки (1. 2. 3.)

### Колонтитулы

- `header1.xml` — шапка с корпоративным логотипом и названием
- `footer1.xml` — нижний колонтитул с нумерацией страниц

## Как редактировать стили

### Способ 1: Через Style Guide

1. Откройте `gpn-template-styleguide.docx` в Microsoft Word
2. Найдите элемент, стиль которого хотите изменить
3. Откройте панель стилей:
   - Windows: `Ctrl+Alt+Shift+S`
   - Mac: `⌘+Option+Shift+S`
4. Кликните правой кнопкой на стиле → **Изменить...**
5. Измените параметры (шрифт, размер, цвет, интервалы)
6. Сохраните документ
7. Скопируйте как `templates/gpn-template.docx`

### Способ 2: Напрямую в шаблоне

1. Откройте `gpn-template.docx`
2. Создайте текст с нужным стилем (например, напишите заголовок)
3. Измените стиль через панель стилей
4. Удалите тестовый текст
5. Сохраните

## Что задаётся в коде (не в шаблоне)

Некоторые параметры пока задаются в `src/styles/gpn-styles.js`:

```javascript
colors: {
  tableHeaderBackground: "0072C6",  // Голубой фон заголовков таблиц
  tableHeaderText: "FFFFFF",        // Белый текст заголовков
}
```

Чтобы изменить эти цвета, отредактируйте файл `gpn-styles.js`.

## Структура Style Guide

Style Guide содержит примеры всех элементов:

1. **Заголовки** — все три уровня
2. **Текст** — обычный, жирный, курсив, моноширинный
3. **Списки** — маркированные и нумерованные
4. **Таблицы**:
   - Таблица терминов
   - Таблица изменений (Как есть / Как будет)
   - Функциональная таблица
5. **Примечания** — блоки :::note
6. **Гиперссылки** — внешние и внутренние

## Советы

- После изменения стилей создайте тестовый документ для проверки
- Сохраняйте резервную копию шаблона перед изменениями
- Если что-то сломалось — используйте оригинальный шаблон из архива
