# 4. Описание требований

## 4.1 Функция с изображением в сценарии

:::function-table{#func-with-image}
function: Форма подписания документа
task: TEST-001
taskUrl: https://jira.example.com/browse/TEST-001
scenario: |
  **Предусловия:**
  - Пользователь авторизован в системе
  - Документ готов к подписанию

  **Основной сценарий:**
  1. Пользователь открывает форму документа
  2. Система отображает форму подписания:

  ![Тестовое изображение](images/test-screenshot.png){width="80%"}

  3. Пользователь нажимает кнопку "Подписать"
  4. Система выполняет подписание

  **Альтернативный сценарий:**
  - При ошибке отображается сообщение
:::

## 4.2 Функция без изображения

:::function-table{#func-without-image}

  **Основной сценарий:**
  1. Пользователь выполняет действие
  2. Система обрабатывает запрос
  3. Результат отображается
:::
